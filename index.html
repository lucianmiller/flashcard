JavaScript and jQuery Flashcards

JavaScript

A programming language used to make web pages interactive

Operators

An operator is a special character (or characters) that indicates an action to be performed.

Variables

A container that stores some information such as a number, an object, or a string.

Variable naming conventions

A set of rules for choosing the character sequence to be used for identifiers which denote variables, types, functions, and other entities in source code and documentation.

Functions

A set of statements that performs a task or calculates a value, but for a procedure to qualify as a function, it should take some input and return an output where there is some obvious relationship between the input and the output.

Methods

An action that a number can perform or have taken on it.

Arguments

An Array-like object accessible inside functions that contains the values of the arguments passed to that function

Parameters

In a function call, are the function's arguments. JavaScript arguments are passed by value: The function only gets to know the values, not the argument's locations. If a function changes an argument's value, it does not change the parameter's original value.

Return

Ends function execution and specifies a value to be returned to the function caller.

Chaining methods

Also known as Cascading, means repeatedly calling one method after another on an object, in one continuous line of code.

Strings

A set of characters enclosed in quotations

Boolean

One of two values: True or False

Undefined

Variable hasn't been defined yet

NaN

Property is a value representing Not-A-Number.

Null

Nothing

Alert

Method displays an alert box with a specified message and an OK button.

Comments

Single line comments start with // . Any text between // and the end of the line will be ignored by JavaScript

jQuery

A JavaScript library that makes it easier to incorporate JavaScript on your web page

Attributes

A collection of objects that belong to a built-in Attr class, with name and value properties.
